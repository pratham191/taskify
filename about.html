About taskify

A comprehensive task management system designed to help you organize your work and boost productivity.

Project Overview

taskify is a task management web application developed as a college project at Indian Institute of Technology, Patna. The project aims to demonstrate the implementation of a responsive web application using HTML, CSS, and JavaScript with client-side data persistence.

Features

  • User registration and authentication
  • Create, edit, and delete tasks
  • Set task priorities and due dates
  • Mark tasks as completed
  • Filter and sort tasks
  • Responsive design for all devices
  • Dark and light theme options

Technologies Used

  • HTML5 - Structure
  • CSS3 - Styling and Animations
  • JavaScript - Functionality and Interactivity
  • LocalStorage - Data Persistence
  • Font Awesome - Icons
